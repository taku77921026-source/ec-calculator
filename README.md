# EC原価・利益計算ツール

EC物販（Amazon / 楽天 / Yahoo! / Shopify）向けの原価計算・利益シミュレーションツール。
仕入れコストを入力すると、1個あたり原価・プラットフォーム別の利益額／利益率・必要販売価格が分かる。

## 主な機能
- 元価格（元）→ 為替レートで円換算（為替はコスト項目で入力）
- 仕入れ数量・関税・消費税・諸費用（送料／検品料／代行手数料／梱包材／同梱物）
- 仕入原価内訳（総額＋1個あたり）
- プラットフォーム別シミュレーション（販売価格→利益 ／ 目標利益率→必要販売価格）
- 手数料率は「設定」タブで管理
- 商品マスター（登録・上書き・削除）＋スプレッドシート自動連携
